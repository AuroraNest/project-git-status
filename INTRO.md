# Git Repo Manager

一款 macOS 原生 Git 仓库管理工具。

**痛点：** Vibe Coding 时代，用 Claude Code / Cursor 等 AI 在命令行写代码，几乎不打开 IDE。但 AI 改了什么心里没底，想 review 还得开编辑器。提交推送？又得敲一堆 git 命令。

**这个工具：** 不用开 IDE、不用敲命令，可视化搞定一切。一眼看到 AI 改了哪些文件、diff 长啥样，确认没问题点一下就 commit + push。菜单栏常驻，随时瞄一眼状态。

**功能：** 多仓库管理 / 文件变更预览 / Diff 对比 / 一键提交推送 / 分支管理 / 提交历史回滚

SwiftUI 原生，轻量快速，中英文支持。
